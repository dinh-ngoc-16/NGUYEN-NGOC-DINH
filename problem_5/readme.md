## Installation

run with nodejs

1. npm i
2. npm start
3. url of api => http://localhost:3000/

---

run with docker

1. docker build -t my-app .
2. docker run -d -p 3000:3000 --name my-running-app my-app
3. url of api => http://localhost:3000/
